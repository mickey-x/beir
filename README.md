# Beir | Backend

Using 'firebase' + 'express' to build the backend of Beir.


### 1- API Base URL
add `https://europe-west3-beir-app.cloudfunctions.net/api` as the 'proxy' value in package.json

### 2- Install packages
run `npm install`

### 3- Run project
run `npm start`

### 4- Open it
go to `http://localhost:3000`
